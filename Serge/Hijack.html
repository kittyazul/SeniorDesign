SESSION HIJACKING


PART I: DEFINITION


In a session hijacking attack, an attacker exploits the web session control mechanism (normally managed for a session token) to steal information from the victim/user. Since a web server needs a method to recognize a user’s connection, a token (similar to a URL) is sent to the client browser after a successful client authentication. By compromising that session token (or connection), an attacker is able to gain unauthorized access into a user's system.


PART II: HOW ARE SESSION HIJACKING ATTACKS CONDUCTED?


To perform a successful session hijacking attack, an attacker needs to know the victim’s session ID (session key). By using a session sniffer sniffer tool, the attacker is able to intercept relevant information. Stealing the session cookie or simply persuading the user to click a malicious link will enable them to hijack the session. The server will then be fooled into treating the attacker’s connection as the original user’s valid session. A session token can also be compromised in several other ways, though the most common are:

  • Client-side attacks (XSS, malicious JavaScript Codes, Trojans, etc);
  • Man-in-the-middle attack;
  • Man-in-the-browser attack.



PART III: WHAT CAN ATTACKERS DO AFTER SUCCESSFUL SESSION HIJACKING?


If successful, an attacker can perform any actions that the original user is authorized to do during the active session. Depending on their permissions and the targeted application, this may mean anything from transferring money from the user’s bank account or posing as them to buy items in online stores, to encrypting valuable data and demanding a ransom to decrypt them. For larger organizations, this is even more dangerous because that cookie can be used to identify authenticated users in Single Sign-On systems (SSO). In other words, the attacker will be able to potentially gain unauthorized SSO access to multiple applications and systems, ranging anywhere from financial systems and customer records to systems containing valuable intellectual property.
